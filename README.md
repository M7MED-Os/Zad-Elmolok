# 👑 زاد الملوك - منيو المأكولات الرقمي ونظام طلبات الواتساب

منيو رقمي فاخر وتفاعلي لمطعم **زاد الملوك** (مشويات - طواجن - صواني - سندوتشات)، مصمم خصيصاً للهواتف المحمولة ليتم فتحه عبر مسح رموز الاستجابة السريعة (QR Codes) في الطاولات أو المواد الدعائية.

يعمل الموقع بالكامل في جانب العميل (Client-Side) بدون قواعد بيانات أو خادم خلفي، مع إتاحة سلة مشتريات تفاعلية، واختيار البدائل (الأحجام)، ونظام إرسال الطلبات المباشر عبر الواتساب مع حفظ بيانات العميل تلقائياً.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

تم بناء المشروع باتباع بنية برمجية خفيفة وسريعة جداً لضمان التحميل الفوري:
- **HTML5** (هيكلية دلالية مهيأة للـ SEO ومحركات البحث).
- **CSS3** (نظام تصميم فاخر وداكن يعتمد على متغيرات CSS مخصصة وتجربة تصفح سلسة ومحسنة للهواتف).
- **Vanilla JavaScript** (إدارة كاملة لحالة السلة والطلبات، والبحث الفوري، والتحقق من البيانات).
- **PWA Capabilities** (دعم كامل للتثبيت والعمل أوفلاين عبر الـ Cache).

## 🚀 خطوت رفع ونشر الموقع على GitHub Pages

الموقع جاهز للنشر المباشر والمجاني على منصة **GitHub Pages**، فقط اتبع الخطوات التالية:

### 1. إنشاء مستودع (Repository) على GitHub
- اذهب إلى حسابك على GitHub وأنشئ مستودعاً جديداً (New Repository).
- اجعل المستودع **عاماً (Public)**.
- قم بتسمية المستودع (مثال: `zad-elmolok`).

### 2. رفع ملفات المشروع باستخدام Git
افتح شاشة الأوامر (Terminal) في مجلد المشروع وقم بتشغيل الأوامر التالية:

```bash
# تهيئة ملفات الـ Git
git init

# إضافة جميع الملفات
git add .

# تسجيل التغييرات
git commit -m "زاد الملوك: النسخة الأولى المستقرة"

# تغيير الفرع الأساسي إلى main
git branch -M main

# ربط المجلد المحلي بالمستودع على جيتهاب (استبدل USERNAME و REPO_NAME ببياناتك)
git remote add origin https://github.com/USERNAME/REPO_NAME.git

# رفع الملفات
git push -u origin main
```

### 3. تفعيل GitHub Pages
1. اذهب إلى صفحة المستودع الخاص بك على موقع GitHub.
2. من القائمة العلوية، اختر **Settings** (الإعدادات).
3. من القائمة الجانبية اليسرى، اختر قسم **Pages**.
4. تحت خيار **Build and deployment**:
   - تأكد من اختيار **Deploy from a branch** كـ Source.
   - تحت قسم **Branch**، اختر الفرع `main` والمجلد `/ (root)`.
5. اضغط على زر **Save** (حفظ).
6. سيقوم جيتهاب ببناء الموقع ونشره تلقائياً خلال دقيقة واحدة، وستظهر لك رسالة تحتوي على الرابط المباشر للموقع في الأعلى بالصيغة التالية:
   `https://USERNAME.github.io/REPO_NAME/`

## 📁 الهيكل التنظيمي للملفات

```text
/
├── index.html            # واجهة الموقع والـ HTML الأساسي
├── manifest.json         # ملف إعدادات تطبيق الويب PWA
├── service-worker.js     # محرك التخزين المؤقت للعمل أوفلاين
├── css/
│   └── styles.css        # نظام التصميم والألوان والخطوط
├── js/
│   ├── data.js           # كود بيانات المطعم وأسعار المنيو
│   └── app.js            # الكود البرمجي لإدارة السلة والطلبات والبحث
└── README.md             # دليل الاستخدام والنشر
```

## ✨ أهم مميزات النظام
- **تحويل أرقام ذكي**: يقوم تلقائياً بمسح الصفر الأول من رقم العميل لتشكيل رابط الواتساب الرسمي الدولي السليم (`2011...`) لمنع أخطاء التحويل والتوجيه.
- **تأكيد الخطوط**: يحتوي على نافذة اختيار (Modal) أنيقة تظهر بعد تعبئة البيانات ليختار العميل إرسال الطلب للخط الأول أو الثاني لتوزيع الحمل وتأمين سرعة الرد.
- **دعم الأوفلاين**: بفضل الـ Service Worker، يمكن للعميل فتح المنيو وتصفحه حتى في حال انقطاع شبكة الإنترنت داخل المطعم بعد التحميل الأول.
- **حفظ تلقائي**: يتم حفظ بيانات العنوان والاسم في الـ LocalStorage لتسهيل إعادة الطلب للعملاء المتكررين.
